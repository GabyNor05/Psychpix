## How to run Monaware

#### Step 1:

Clone the repo:
```
https://github.com/andrevanheerden/formative-one-Monaware.git](https://github.com/GabyNor05/Psychpix.git
```

#### Step 2:

open psychpix backend in terminal:

```
cd backend
```

#### Step 3:

Run Backend:

```
npm start
```
#### Step 4:

Open a new terminal:

#### Step 5:

open psychpix frontend in terminal:

```
cd psychpix
```

#### Step 6:

Run Frontend:

```
npm start
```
